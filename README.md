# SDC PCB
Shudown Circuit main PCB of Formula Student team "Black Hornets" from the University of Novi Sad Faculty of Technical Sciences

Inputs:
BSPD: BSDP_Out
AMS: DISCHARGE_ENABLE
IMD: OkHS
TSAL: Error_Shutdown
LVMS: SDC_IN
Reset button: RESET
LVB: 12V

Outputs:
Dahboard: IMD
Dahboard: AMS
Shutdown Line: SDC_OUT
